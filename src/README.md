# 🚀 Funcionalidades

- Geração automática de relatório fotográfico
- Geração automática do Anexo C
- Organização automática das imagens
- Cálculo de áreas inspecionadas
- Interface gráfica em CustomTkinter
- Exportação em PDF
- Compatível com inspeções industriais

---

# 🛠 Tecnologias

- Python
- CustomTkinter
- ReportLab
- Pillow
- PyPDF2

---

# ▶ Como executar

```bash
pip install -r requirements.txt

python src/interface.py
```
