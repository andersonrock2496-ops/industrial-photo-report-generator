# Relatório Fotográfico Automático

Sistema em Python para gerar relatórios fotográficos em PDF a partir de uma pasta de imagens.

## Funcionalidades

- Seleção de pasta com fotos
- Geração automática de PDF
- Legendas baseadas no nome do arquivo
- Identificação de status por código no nome da foto
- Interface gráfica com CustomTkinter

## Tecnologias

- Python
- ReportLab
- Pillow
- CustomTkinter

## Como executar

```bash
pip install -r requirements.txt
python interface.py
